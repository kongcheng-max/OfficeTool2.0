## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **专业信赖** | 企业级产品，视觉沉稳、专业、可信赖 |
| **效率优先** | 减少操作步数，高频操作一键直达 |
| **反馈明确** | 每个操作有状态反馈（加载、成功、失败） |
| **渐进披露** | 复杂功能默认隐藏，按需展开 |
| **可访问性** | 色盲友好、键盘可操作、读屏器兼容 |

**交互要点**:
- AI 回答支持 **SSE 流式逐字展示**（打字机效果）

- 答案底部：来源文档链接（点击展开原文）+ 置信度
- 支持追问：保持多轮对话上下文
- 空状态：提示输入问题 / 建议问题示例

## 6. 交互规范

### 6.1 通用交互

| 场景 | 行为 |
|------|------|
| **加载** | 表格/列表使用骨架屏 (Skeleton)；按钮 loading 状态 |
| **空状态** | 显示 Empty + 引导操作按钮 |
| **错误** | Toast 提示 + 错误详情可展开 |
| **成功** | Toast 提示 (3s 自动消失) |
| **删除** | 二次确认 (Popconfirm) |
| **长操作** | 进度条 + 预计剩余时间 |
| **网络异常** | 全局网络状态监听 + 断线提示 |

### 6.2 对话交互流程

```
用户进入问答页面
      │
      ▼
┌─────────────┐
│  选择知识库   │ ← 顶部下拉 / URL 参数带入
└──────┬──────┘
       │
       ▼
┌─────────────┐     ┌──────────────┐
│  输入问题     │────→│ 发送到 API   │
│  Enter 发送  │     │ POST /qa/stream│
└─────────────┘     └──────┬───────┘
                           │
                           ▼
                    ┌──────────────┐
                    │ SSE 流式接收  │
                    │ 逐字渲染答案  │
                    └──────┬───────┘
                           │
                           ▼
                    ┌──────────────┐
                    │ 渲染引用来源  │
                    │ + 置信度      │
                    └──────┬───────┘
                           │
                           ▼
                    ┌──────────────┐
                    │ 用户可以追问  │ ← 多轮对话 (Phase 2)
                    │ 或新问题      │
                    └──────────────┘
```

### 6.3 文档上传交互流程

```
用户点击上传 / 拖拽文件
      │
      ├── 前端校验：格式 + 大小
      │   ├── 不通过 → 提示错误
      │   └── 通过 → 开始上传
      │
      ▼
  上传进度条 (百分比 + 速度)
      │
      ▼
  上传完成 → 显示「解析中...」状态
      │
      ▼
  Celery 异步处理
      │
      ├── 轮询 / WebSocket 获取状态
      │
      ▼
  ✅ 就绪  → 自动刷新文档列表
  ❌ 失败  → 显示错误信息 + 重试按钮
```

## 7. 响应式策略

### 7.1 断点定义

| 断点 | 宽度 | 适配 |
|------|------|------|
| Desktop | ≥ 1200px | 完整布局：侧边栏 + 内容 |
| Tablet | 768px - 1199px | 折叠侧边栏（图标模式）/ 可展开 |
| Mobile | < 768px | 无侧边栏；底部 Tab 导航 |

### 7.2 各页面适配

| 页面 | Desktop | Tablet | Mobile |
|------|---------|--------|--------|
| 对话问答 | 对话列表 + 对话区 | 仅对话区 + 汉堡菜单 | 全屏对话 + 底部输入 |
| 知识库卡片 | 3-4 列网格 | 2 列网格 | 单列 |
| 文档表格 | 完整表格 | 隐藏部分列 | 列表视图（卡片化） |
| 图谱可视化 | 完整交互 | 简化控制 | ❌ 不支持（提示用桌面端） |
| 管理后台 | 完整表格 | 简化版 | ❌ 不支持 |
