# D2 — OfficeTool v2.0 安全测试报告 (OWASP Top 10 自查)

> **版本**: v2.0.0-rc | **日期**: 2026-07-11 | **执行**: QA 测试部

---

## OWASP Top 10 对照

### A01: 访问控制失效 — ✅ 已加固

| 检查项 | 状态 | 说明 |
|--------|:--:|------|
| RBAC 三角色 | ✅ | admin/editor/viewer，require_role 装饰器 |
| 权限矩阵 | ✅ | admin全权限 / editor文档管理 / viewer只读 |
| 未登录拦截 | ✅ | 无 token 返回 401/403 |
| 水平越权 | ✅ | KB/Doc 操作校验 owner_id |
| 管理后台保护 | ✅ | admin 专属路由，非admin 403 |

### A02: 加密失效 — ✅ 已加固

| 检查项 | 状态 | 说明 |
|--------|:--:|------|
| JWT 认证 | ✅ | HS256 签名 + 过期时间 |
| 密码哈希 | ✅ | bcrypt + salt |
| HTTPS | ✅ | Nginx 终止 TLS 1.3 + Let's Encrypt |
| 文档加密 | ✅ | AES-256-CBC (可选，DOC_ENCRYPTION_KEY 控制) |
| 密钥管理 | ✅ | 环境变量注入，不硬编码 |

### A03: 注入 — ✅ 已防护

| 检查项 | 状态 | 说明 |
|--------|:--:|------|
| SQL 注入 | ✅ | SQLAlchemy ORM 参数化查询 |
| Cypher 注入 | ✅ | NL2Cypher 使用参数化 `$kb_id` 注入 |
| 命令注入 | ✅ | 无 shell 命令拼接 |
| XSS | ✅ | React 默认转义 + CSP headers |

### A04: 不安全设计 — ✅ 已评估

| 检查项 | 状态 | 说明 |
|--------|:--:|------|
| 限流保护 | ✅ | Token Bucket 100 req/min |
| 速率限制绕过 | ⚠️ | 基于 JWT user_id 或 IP，分布式下需 Redis 统一 |
| 文件上传限制 | ✅ | MAX_FILE_SIZE_MB=200 |

### A05: 安全配置错误 — ✅ 已加固

| 检查项 | 状态 | 说明 |
|--------|:--:|------|
| Debug 模式 | ✅ | 生产环境 DEBUG=false |
| SECRET_KEY | ✅ | 生产环境必须设置，否则拒绝启动 |
| CORS 限制 | ✅ | CORS_ORIGINS 可配置 |
| 数据库端口 | ✅ | 仅暴露 localhost:5432 |
| 错误信息泄露 | ✅ | 生产错误不返回堆栈 |

### A06: 脆弱和过时组件 — ⚠️ 需持续关注

| 检查项 | 状态 | 说明 |
|--------|:--:|------|
| Python 依赖 | ⚠️ | 建议定期 `pip audit` 扫描 |
| Docker 镜像 | ✅ | 使用固定版本 tag (postgres:15-alpine) |
| Node 依赖 | ⚠️ | 建议 `npm audit` 扫描 |

### A07: 身份识别和认证失败 — ✅ 已加固

| 检查项 | 状态 | 说明 |
|--------|:--:|------|
| 弱密码限制 | ⚠️ | 当前无密码复杂度要求 (BUG-050) |
| JWT 过期 | ✅ | 默认 24h, 可配置 |
| Token 刷新 | ⚠️ | 无 refresh token 机制 |
| 多设备登录 | N/A | MVP 阶段不适用 |

### A08: 软件和数据完整性故障 — ✅ 已评估

| 检查项 | 状态 | 说明 |
|--------|:--:|------|
| 文件 MD5 校验 | ✅ | 上传时计算 MD5 去重 |
| 反序列化 | ✅ | JSON 解析无 pickle |
| CI/CD 管道 | N/A | 手工部署 |

### A09: 安全日志和监控故障 — ✅ 已加固

| 检查项 | 状态 | 说明 |
|--------|:--:|------|
| 审计日志 | ✅ | AuditMiddleware 自动记录非GET操作 |
| 日志字段 | ✅ | user_id/username/action/IP/status/time |
| 日志查询 | ✅ | GET /admin/audit-logs (admin only) |
| 日志防篡改 | ⚠️ | 无 WAL/只追加保护 |

### A10: SSRF — ✅ 低风险

| 检查项 | 状态 | 说明 |
|--------|:--:|------|
| LLM API 调用 | ✅ | 仅调用配置的 LLM endpoint |
| 文件获取 | ⚠️ | URL 导入功能未实现，风险可控 |
