## 五、已知限制

| 限制 | 影响 | 备注 |
|------|------|------|
| 图谱在手机端交互受限 | 小屏幕 | G6 力导向图在小屏上难以操作，建议平板以上 |
| Safari WebKit 下 SSE 偶发断连 | 流式问答 | 浏览器 Bug，非代码问题 |
| IE 11 不支持 | 企业遗留 | React 18 不兼容 IE |
| OCR 需 Tesseract 二进制 | 图片解析 | 需在 Dockerfile 中安装 `tesseract-ocr` |

## 结论

✅ **Chrome / Edge / Firefox / Safari 四浏览器兼容**。移动端 768px 以下汉堡菜单适配。生产环境 Docker Compose 全量编排支持 Ubuntu/Debian/CentOS。图谱和 OCR 有已知设备限制，不影响 GA 发布。
